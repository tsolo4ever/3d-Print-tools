## üöÄ **INSTALLATION INSTRUCTIONS**

### SuperSlicer Installation:

1. **Locate your SuperSlicer config folder:**
   - Windows: `%APPDATA%\SuperSlicer\`
   - Mac: `~/Library/Application Support/SuperSlicer/`
   - Linux: `~/.config/SuperSlicer/`

2. **Install Printer Profiles:**
   - Copy 4 printer .ini files to `printer/` subfolder
   - Files: `Creality_Ender-5_Plus_[size]mm_Main_CORRECTED.ini`

3. **Install Print Profiles:**
   - Copy all 25 print profile .ini files to `print/` subfolder
   - Files: `[layer_height]mm_[quality]_@DD.ini`

4. **Restart SuperSlicer**

5. **Select Profiles:**
   - Printer: Choose your nozzle size (0.4mm, 0.6mm, 0.8mm, or 1.0mm)
   - Print: Choose your quality level
   - Filament: Use your existing filament profiles

## üí° **USAGE TIPS**

### Switching Between Profiles:

1. **When changing nozzle size:**
   - Physically change nozzle on printer
   - Adjust Z-offset (baby-step during first print)
   - Select matching printer profile in slicer
   - Select appropriate print quality profile

2. **When changing quality level (same nozzle):**
   - Just select different print profile
   - Everything else stays the same

### Profile Selection Strategy:

**Ask yourself:**
1. **Does this part need fine detail?**
   - YES ‚Üí Use DETAIL or HIGHDETAIL profile
   - NO ‚Üí Continue to question 2

2. **Do I need this fast?**
   - YES ‚Üí Use DRAFT or SUPERDRAFT profile
   - NO ‚Üí Use OPTIMAL or NORMAL profile

3. **Is this a one-time print or production?**
   - One-time ‚Üí Can use slower, higher quality
   - Production ‚Üí Use fastest acceptable quality

## üîç **TROUBLESHOOTING BY PROFILE**

### HIGHDETAIL Profiles (0.08-0.15mm):

**Problem:** Taking forever to print
- **Solution:** This is normal; consider DETAIL profile instead

**Problem:** Nozzle clogging
- **Solution:** Ensure filament is dry, reduce retraction slightly, verify extruder tension

**Problem:** Poor first layer adhesion
- **Solution:** Reduce first layer speed to 15 mm/s, increase bed temp +5¬∞C

---

### OPTIMAL/NORMAL Profiles (0.16-0.40mm):

**Problem:** Layer lines visible
- **Solution:** This is normal; use finer profile if critical

**Problem:** Stringing between parts
- **Solution:** Increase retraction +0.5mm, reduce temp -5¬∞C

**Problem:** Top surface not smooth
- **Solution:** Increase top solid layers +1, reduce top infill speed -10 mm/s

---

### DRAFT/SUPERDRAFT Profiles (0.42-0.80mm):

**Problem:** Very visible layer lines
- **Solution:** Expected; use finer profile or sand/acetone smooth

**Problem:** Gaps in top layers
- **Solution:** Increase top layers +1, add 5-10% flow

**Problem:** Weak overhangs
- **Solution:** Use supports, reduce overhang speed to 15 mm/s

## üéì **PROFILE NAMING EXPLAINED**

Format: `[layer_height]mm_[quality_level]_@DD.ini`

- **Layer height:** Actual layer thickness in mm
- **Quality level:** Subjective quality descriptor
  - HIGHDETAIL: Maximum detail possible
  - DETAIL: Good detail, reasonable speed
  - OPTIMAL: Best balance speed/quality ‚≠ê
  - NORMAL: Standard everyday printing
  - FAST: Prioritizes speed over detail
  - DRAFT: Quick prototypes, visible layers
  - SUPERDRAFT: Maximum speed, rough surface

- **@DD:** Indicates "Direct Drive" optimized
  - Retraction distances appropriate for DD
  - Conservative speeds for reliability
