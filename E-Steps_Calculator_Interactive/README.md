# E-Steps Calculator Interactive

An interactive web-based calculator for calibrating 3D printer extruder steps (E-steps).

## Purpose
This tool helps 3D printer users accurately calibrate their extruder steps per millimeter (steps/mm) to ensure proper filament extrusion.
